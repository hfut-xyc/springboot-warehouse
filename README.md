#### 开发教程
1. 安装Java 8+、Maven、node.js和npm
2. npm安装@vue/cli和vue的开发服务器
3. 安装MariaDB/MySQL，配置root用户密码为root，服务器端口号为3306
4. 后端启动mvn spring-boot:run
5. 前端启动npm run serve
